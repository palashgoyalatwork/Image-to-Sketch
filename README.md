# Image to Sketch Generator

A Python application that converts a normal image into a pencil sketch using OpenCV.

## Features
- Convert images to sketch
- Simple and beginner-friendly
- Fast processing

## Technologies Used
- Python
- OpenCV

## How to Run

1. Install OpenCV:
   ```bash
   pip install opencv-python
   ```

2. Run:
   ```bash
   python app.py
   ```
